# Node, Express and APIs
- Node.js is an event-based, non-blocking, asynchronous I/O runtime that uses Google's V8 JavaScript engine and libuv library
V8 engine is the open-source JavaScript engine- creator Ryan Dahl took the V8 engine and enhanced it with features like file system API, HTTP library and a operating system related utility methods.
- excellent support for modern JavaScript, you can wite using the latest and most modern syntax and generally don't have to worry about compatibility issues.
- Node.js lets you run JS on the server
- Execution model causes the server very little overhead and capable of handling a large number of simltaneous connections.
- suited to building applications that require some form of real-tim interaction or collaboration= chat sites, CodeShare where you watch a cod being edited live. Also building APIs handling a lot of requests
- advantages from speed and scalability, your brain no longer needs  to switch modes you can do everything in the same language
- It speaks JSON, the most important data exchange on the web
